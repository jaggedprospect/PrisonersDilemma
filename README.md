# Prisoner's Dilemma

A variation of the Prisoner's Dilemma game meant to be played from the Terminal.

Written in C#.

### Installation

Download the zipped `Project_0` file and extract..

OR

..Copy the source code and txt files into your own .NET project.

### Usage

If you downloaded and extracted the project folder, simply navigate to `Project_0` in your Terminal and run it via `dotnet run` .

If you copied the files into your own project folder, make sure to run these commands to setup and run your project:

`dotnet new console` (Skip this step if you've already setup .NET for your project) 

`dotnet build` (Builds the project)

`dotnet run` (Runs the project)
